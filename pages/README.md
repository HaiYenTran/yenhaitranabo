# Cấu trúc nội dung (restructured)

Thư mục `pages/` gom toàn bộ trang nội dung theo nhóm để dễ tìm và quản lý:

- `pages/stories/`: chuỗi bài viết `nhung-cau-chuyen-*` và bài liên quan.
- `pages/breakfast/`: nhóm bài về bữa sáng.
- `pages/digital/`: nhóm bài chương trình digital.
- `pages/passive-income/`: nhóm bài passive income.
- `pages/health/`: nhóm bài sức khỏe và lifestyle.
- `pages/showcase/`: nhóm trang demo, gallery, minh họa.

> Các file HTML ở thư mục gốc vẫn được giữ để **redirect tự động** sang vị trí mới,
> giúp các link cũ vẫn hoạt động.
